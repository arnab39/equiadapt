# Commands to run N-Body Model
```
python examples/nbody/train.py
```
# Changing Canonicalizer
Currently only vndeepsets is supported. To change hyperparameters, In `examples/nbody/train.py`, change the values in `CANON_HYPERPARAMS`.

# Changing Prediction
In `examples/nbody/train.py`, change the values in `PRED_HYPERPARAMS`.
